# PlayReserve

Plateforme web de réservation de terrains sportifs en ligne.

## Description

PlayReserve permet aux clubs sportifs de gérer leurs terrains et créneaux horaires, et aux clients de réserver facilement un terrain et d’effectuer un paiement en ligne. Le système inclut trois types d’utilisateurs : super administrateur, administrateur de club et client.

## Fonctionnalités principales

- Gestion des comptes utilisateurs (inscription, connexion, rôles)
- Gestion des clubs (ajout, modification, suppression)
- Gestion des terrains par club
- Gestion des créneaux horaires
- Réservation de terrains avec vérification de disponibilité
- Paiement en ligne (simulé)
- Tableaux de bord par rôle (admin global, admin club, client)
- Historique des réservations et paiements

## Technologies utilisées

- Laravel (PHP)
- MySQL
- Tailwind CSS (via Laravel Breeze)
- Git
